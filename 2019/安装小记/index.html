一、Vue（v3.9.2）

  1. 安装 vue

    $ npm|cnpm install vue

  2. 安装 vue-cli

    #npm 全局安装
$ npm|cnpm install -g @vue/cli
#OR
#Yarn 全局安装
$ yarn global add @vue/cli

  3. 查看版本号

    $ vue --version
#OR
$ vue -V

  4. 查看 vue 安装路径

    $ which vue

  5. 卸载 vue-cli

    #npm 全局卸载
$ npm|cnpm uninstall -g @vue/cli
#OR
#yarn 全局卸载
$ yarn global remove @vue/cli




二、Webpack（v4.36.1)

  1. 新建 webpack 项目

    $ mkdir webpack-name
$ cd webpack-name

  2. 创建 package.json 文件

    $ npm init -y

  3. 本地安装（推荐）

    #安装最新版本
$ npm install --save-dev webpack
#安装特定版本
$ npm install --save-dev webpack@<version>
#webpack V4+ 版本需安装cli
$ npm install --save-dev webpack-cli

  4. 本地卸载

    #卸载webpack-cli
$ npm un webpack-cli
#卸载webpack
$ npm un webpack

  5. 删除 webpack 残余文件

    #node_modules
#package.json
#package-lock.json

  6. 全局安装

    $ npm install --global webpack
#简写
$ npm install -g webpack

  7. 全局卸载
    $ npm uninstall –global webpack

    #简写
$ npm un -g webpack




三、更新PHP(没成功)

  1. 查看PHP版本

    $ php -v
//PHP 7.1.23 (cli)

  2. 查看PHP路径

    $ whereis php

  3. 跟新PHP(7.3为例)

    $ curl -s https://php-osx.liip.ch/install.sh | bash -s 7.3

  4. 配置路径

    $ vim ~/.bash_profile

  5. 添加如下

    export PATH=/usr/local/php5/bin:$PATH;

  6. 使配置生效

    $ source ~/.bash_profile

  7. 参考链接

    PHP官网




四、Homebrew

  1. 更新homebrew

    $ brew update
//Updating Homebrew...
//==> Auto-updated Homebrew!

  2. 用Homebrew下载PHP

    $ brew install php

  3. 用Homebrew更新PHP

    $ brew upgrade php

  4. 用Homebrew卸载PHP

    $ brew uninstall php
