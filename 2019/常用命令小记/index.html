一、Git

  1. 查看分支

    $ git branch      //查看本地
$ git branch -r   //查看远程
$ git branch -a   //查看全部

  2. 新建分支

    $ git branch xxx

  3. 切换分支

    $ git checkout xxx

  4. 新建并切换分支

    $ git checkout -b xxx

  5. 删除分支

    $ git branch -d xxx     //删除本地分支
$ git push origin :xxx     //删除远程分支
$ git push origin xxx     //提交分支

  6. 拉取远程最新代码

    $ git pull origin xxx   //xxx为分支，可mas-ter可任意

  7. 提交本地最新代码

    $ git status      //查看状态，即修改了什么
$ git add -A      //添加所有修改内容到仓库
$ git commit -m "解释说明"    //提交内容到仓库，并解释说明
$ git push origin xxx   //提交到远程xxx分支

  8. 查看提交历史

    $ git log 
$ git log -p <file> //产看指定文件的提交历史


二、终端

  1. 目录

    $ cd  
$ ls 
$ ls -la   
$ pwd   //打印当前目录
$ mkdir xxx //创建目录
$ rmdir xxx     //删除目录
$ mvdir xxx

  2. 文件

    $ open xxx  //使用默认程序打开
$ cp    //复制
$ vim name.text //新增修改
$ rm    //删除
$ mv    //改变


三、Git 提交注释

  1. [feat]:新功能

  2. [fix]:修复bug

  3. [add]:添加某些内容

  4. [remove]:删除某些内容

  5. [update]:更新某些内容

  6. [refactor]:重构(不是新功能、修复bug)

  7. [style]:格式

  8. [chore]:琐事

  9. [docs]:文档

  10. [test]:测试


四、SSH 连接远程服务器

连接服务器需要知道服务器IP、端口号、服务器的账号和密码:

  1. 打开终端,进入根目录

    $ sudo su -             //进入root目录
$ password: xxxxx       //输入电脑账户密码

  2. 连接

    $ ssh -p 端口号 用户名@IP       //如 ssh -p 19222 web@10.1.63.167
$ yes                         // 确认连接
$ password: xxxxx             //输入服务器账户密码

  3. 连接成功
