基本数据类型

一、typeof操作符

检测给定变量的数据类型，将返回下列某个字符串：

  • "undefined" ____值未定义
  • "boolean" ____布尔值
  • "string" ____字符串
  • "number" ____数值
  • "object" ____对象|null
  • "function" ____函数

typeof操作符可以是变量，也可以是数组字面量。typeof不是一个函数，圆括号不是必需的。例：

var message = "some string";
console.log(typeof(message));   //string
console.log(typeof message);    //string
console.log(typeof 95);         //number


二、undefined类型

变量只声明未定义时，值为undefined。

var message;
//  var age;
//  console.log(age);            //报错
console.log(message);           // undefined
console.log(typeof message);    //undefined
console.log(typeof age);        //undefined


三、null类型

逻辑角度，null值表示一个空对象指针，所以typeof检测null值时返回object。保存对象的变量还没有真正的保存对象时，就应该明确的让该变量保存null值。


四、Boolean类型

两个字面量：true、false；与数字值不是一回事，true不一定等于1，false不一定等于0；且区分大小写。

Boolean()：将一个值转换为对应的Boolean值

转换规则对理解流控语句【if语句、while语句等】自动执行相应的Boolean转换非常重要。

数据类型转换为true值转换为false值

Number任何非0数字值(包括无穷大)0、NaN
Object任何对象null
undefinedN/Aundefined


五、Number类型

数值范围[ -Infinity, Infinity ]

想要确定一个数值是不是有穷，即是不是位于最小和最大之间，可以使用isFinite()函数:

var result = Number.MAX_VALUE + Number.MIN_VALUE;
console.log(isFinite(result));          //true


NaN:非数值（Not a Number）

用于表示一个本来要返回数值的操作数未返回数值的情况（这样不会抛出错误）。

两个特点：

  1. 任何涉及NaN的操作（NaN/10、10/NaN）都会返回NaN；在其他编程语言，任何数值除以非数值都会导致错误。

  2. NaN与任何值都不相等，包括NaN本身。

isNaN():接收一个任何类型的参数，判断参数是否“不是数值”；会尝试将参数转换为数值，将不是数值的值直接转换为数值，任何不能被转换为数值的值都会返回true；

console.log(isNaN(NaN));    //true
console.log(isNaN(10));     //false_10是数值
console.log(isNaN('10'));   //false_可以转换成数值10
console.log(isNaN("blue")); //true_不能转换成数值
console.log(isNaN(true));   //false_可以被转换成数值1


数值转换:非数值转换成数值

Number()用于任何数据类型；parseInt()和parseFloat()专门用于把字符串转换成数组；同样的输入会有不同的结果。

Number()转换规则：

  • 数值，简单传入返回;

  • Boolean值，true->1,false->0;

  • null->返回0;

  • undefined->返回NaN;

  • string，遵循规则如下：

    • 只包含数字->转换为十进制数值，’1’->1,’01’->1;

    • 包含有效浮点格式->对应的浮点数值（前导0被忽略）；

    • 包含有效十六进制格式->转换为相同大小的十进制整数值；

    • 字符串为空->0;

    • 包含上述格式之外的字符->NaN.

  • 对象，调用valueOf(),依照上面规则转换返回的值；若未NaN，则调用toString(),依照前面规则转换返回的字符串值。


parseInt():忽略字符串前面的空格，直至找到第一个非空字符。如果第一个字符不是数字或者负号，返回NaN；因此，parseInt()转换空字符串返回NaN（Number()对空字符串返回0）。提供第二个参数：转换时使用的基数。

console.log(parseInt("10",2));  //2  按二进制解析
console.log(parseInt("10",8));  //8  按八进制解析
console.log(parseInt("10",10)); //10  按十进制解析
console.log(parseInt("10",16)); //16  按十六进制解析


parseFloat():忽略前导0，解析直至遇到一个无效的浮点数字字符为止；①只有第一个小数点有效，第二个小数点无效；②十六进制格式字符串始终会被转换成0；③若字符串包含一个可解析的整数（没有小数点，或者小数点后都是0)，返回整数：

console.log(parseFloat("22.34.5"));     //22.34 
console.log(parseFloat("0908.5"));      //908.5
console.log(parseFloat("0xA"));         //0   
console.log(parseFloat("1234bule"));    //1234  
console.log(parseFloat("1234.000"));    //1234  
console.log(parseFloat("22.5"));        //22.5  
console.log(parseFloat("3.215e7"));     //32150000


六、String类型

string类型包含一些特殊的字符字面量-转义序列；用于表示非打印字符或其他用途；字符串一旦创建值是不可变得，要改变某个变量保存的字符串，首先要销毁原来的字符串，然后再用另一个包含新值得字符串填充该变量；

转换为字符串

把一个数字转为一个字符串有两种方式。①toString():返回相应值得字符串表现;数值、布尔值、对象、字符都有toString()方法并返回字符串的一个副本，但null和undefined没有toString()方法。

console.log((11).toString());   //字符串"11"
console.log((true).toString());   //字符串"true"

多数情况下，不用传参，但在调用数值的toString()方法时，可以传递一个参数：输出数值的基数；

console.log((11).toString());   //十进制字符串"11"
console.log((11).toString(8));   //八进制字符串"13"
console.log((11).toString(16));   //十六进制字符串"b"

②String():在不知道要转换的值是null或者undefined的情况下可以使用，能将任何类型的值转换为字符串；

  1. 值有toString()方法，调用该方法并返回相应结果；
  2. null，返回”null”;
  3. undefined,返回”undefined”;

二、 Array类型

①创建

Arrary构造函数

var hotyan = new Array(3);    //创建一个包含3项的数值

数组字面量表示法

var hotyan = []; //创建一个空数组
var hotyan = ["a","b","c"]; //创建一个包含三个字符串的数组


length属性

不是只读，可以从数组的末尾移除项或向数组添加新项；将length的属性设置为小于数组项数的值，被移除的项显示为undefined；反之，新增的项也是undefined；由于数组最后一项的索引始终是length-1，因此下一个新项的位置就是length：

var colors = ['red','blue','green'];    //创建一个包含3个字符串的数组
colors[colors.length] = 'black';    //在数组最后，即位置4添加新项

⑤队列方法

push()：向数组末端添加项，返回修改后 数组长度

例：在数组 arr 中，查找值与 item 相等的元素出现的所有位置

function findAllOccurrences(arr, target) {
    var Arr =[];
    for(var i = 0;i<arr.length;i++)
        if(arr[i]==target)
            Arr.push(i);
    console.log(Arr);
    return Arr;
}
findAllOccurrences('abcdefabc','a');   // [0, 6]


pop()：移除数组末端项并返回 被删除的项

shift()：移除数组第一项并返回 该项，数组长度减1


unshift()：在数组前端添加任意个项并返回 新数组长度

例：在数组 arr 开头添加元素 item。不要直接修改数组 arr，结果返回新的数组：

function prepend(arr, item) {
    var Arr = arr.concat();     //复制一个新数组
    Arr.unshift(item);      //开头添加一位后返回
    return Arr;  
}
prepend([1,2,3,4],10);      //[10,1,2,3,4]


⑥重排序方法

var arr = [1,2,3,4,5];
arr.reverse();

var arr = [0,1,5,10,15];
arr.sort();

compare()：比较函数

function compare(a,b){
    if(a < b){
        return -1;
    }   
    else if(a > b){

    }     
    else{
        return 0;

var arr = [0,1,5,10,15];
arr.sort(compare);
alert(arr);     //0,1,5,10,15_正确

⑦操作方法

concat()：基于当前数组的所有项创建一个新数组

先创建当前数组的一个副本，将接收到的参数添加到这个副本末尾。没有参数即复制当前数组内容并返回副本，传递的参数可以不是数组、一个或多个数组。

var a = ["A","B","C","D"];
var b = a.concat("a",["b","c"]);

alert(a);       //A,B,C,D
alert(b);       //A,B,C,D,a,b,c

例：在数组 arr 末尾添加元素 item。不要直接修改数组 arr，结果返回新的数组

function append(arr, item) {
    return arr.concat(item);
}
append([1,2,3,4],10);//[1, 2, 3, 4, 10]

例：合并数组 arr1 和数组 arr2。不要直接修改数组 arr，结果返回新的数组：

function concat(arr1, arr2) {
    return arr1.concat(arr2);
}


slice(起始位置，结束位置)：基于当前数组中的一或多个项创建一个新数组

可以接收两个参数（要返回项的起始位置，结束位置）；

  • 一个参数时：返回该参数指定位置到当前数组末尾的所有项；
  • 两个参数时：返回起始位置和结束位置（不包括结束位置）之间的项。
  • 若参数为负，用数组长度加上该数来确定相应的位置；
  • 若结束位置小于起始位置，返回空数组。

例：删除数组 arr 最后一个元素。不要直接修改数组 arr，结果返回新的数组：

function truncate(arr) {
    return arr.slice(0,(arr.length)-1);
}
truncate([1,2,3,4]);    //[1, 2, 3]

例：删除数组 arr 第一个元素。不要直接修改数组 arr，结果返回新的数组

function curtail(arr) {
    return arr.slice(1);//返回除第一个的新数组
}


splice()：主要用途是向数组的中部插入项

  • 删除：删除任意数量的项，接收两个参数（要删除的第一项的位置，删除项数）
  • 插入：向指定位置插入任意数量的项，需提供三个参数（起始位置，0：要删除的项数，要插入的项）
  • 替换：向指定位置插入任意数量的项，且同时删除任意数量的项，三个参数（起始位置，要删除的项数，要插入的任意数量的项）

例：在数组 arr 的 index 处添加元素 item。不要直接修改数组 arr，结果返回新的数组：

function insert(arr, item, index) {
    var Arr = arr.concat();
    Arr.splice(index-1,0,item);
    console.log(Arr);
    return Arr;
}
insert([1,2,3,4],'z',2);    //


⑧位置方法

都接收两个参数【要查找的项,（可选）查找起点位置的索引】；都返回要查找的项在数组中的位置，没有找到的情况下返回-1；

indexOf()：从数组的开头（位置0）开始向后查找。返回位置。

例：找出元素 item 在给定数组 arr 中的位置,如果数组中存在 item，则返回元素在数组中的位置，否则返回 -1：

function indexOf(arr, item) {
    return arr.indexOf(item);
}
indexOf([1,2,3,4],3);   //2

例：找出数组 arr 中重复出现过的元素

function duplicates(arr) {
    var Arr = [];
    for(var i = 0;i<arr.length;i++)
        for(var j = i+1;j<arr.length;j++)
            if((arr[i]==arr[j])&&(Arr.indexOf(i)==-1))
                Arr.push(arr[i]);      
    return Arr.sort();
}
duplicates([1, 2, 4, 4, 3, 3, 1, 5, 3]);  //[ 1, 3, 4 ]

lastIndexOf()：从数组的末尾开始向前查找。


⑨迭代方法

最相似的是every()和some()，都用于查询数组中的项是否满足某个条件；filter()用于查询满足某个条件的所有数组项；map()用于创建包含项与另一个数组一一对应的数组；forEach()本质上与for循环迭代数组一样。均不影响原数组。

every()：对数组中每一项运行给定函数，函数对每一项都为true时 返回true

some()：对数组中每一项运行给定函数，函数对任一项为true时 返回true

filter() ：对数组中每一项运行给定函数，返回函数会返回true的项组成的数组

例如：移除数组 arr 中的所有值与 item 相等的元素。不要直接修改数组 arr，结果返回新的数组

function remove(arr, item) {
    return arr.filter(function(items,index,arr){
        return (items!=item);
    });
}
remove([1,2,3,4,2],2);      //[1,3,4]

例如：移除数组 arr 中的所有值与 item 相等的元素，直接在给定的 arr 数组上进行操作，并将结果返回：

function removeWithoutCopy(arr, item) {
    for(var i = 0;i < arr.length;i++){
        if(arr[i] == item){
            arr.splice(i,1);
            i--;
        }
    }
    return arr;
}
removeWithoutCopy([1,2,2,3,4,2,2],2);//


forEach() ：对数组中每一项运行给定函数，没有返回值

例：统计数组 arr 中值等于 item 的元素出现的次数

function count(arr, item) {
    var count = 0;
    arr.forEach(function(items,index,arr){
        if(items == item)
            count+=1;
    });
    console.log(count);
    return count;
}
count([1,2,4,4,3,4,3],4);//3


map() ：对数组中每一项运行给定函数，返回每次函数调用的结果组成的数组

例：为数组 arr 中的每个元素求二次方。不要直接修改数组 arr，结果返回新的数组

function square(arr) {
    return arr.map(function(item,index,arr){
        return item*item;  
    });
}
square([1, 2, 3, 4]);//[1, 4, 9, 16]


10.归并方法

都会迭代数组的每一项，构建一个最终返回的值；用于求数组中的所有值之和；方法都接收两个参数（在每一项上调用的函数，可选的作为归并基础的初始值）；传给reduce()和reduceRight()的函数接收4个参数（前一个值，当前值，项的索引，数组对象）。

reduce()：从第一项开始遍历

例如：计算给定数组 arr 中所有元素的总和：

function sum(arr) {
    return arr.reduce(function(prev,cur,index,arr){
        return prev+cur;
    });
}
sum([1,2,3,4]);
