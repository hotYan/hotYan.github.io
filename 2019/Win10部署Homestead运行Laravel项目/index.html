Win10部署Homestead运行Laravel项目

链接自取

  • laravel 社区




一、准备工作

1. 默认已安装GitBash

  • 否？git for windows


2. 默认已安装Node.js

  • 否？Node.js download




二、部署Homestead运行环境(建议存非C盘)

1. 安装 VirtualBox

  • 选择版本:Windows主机 VirtualBox


2. 安装 Vagrant

  • 选择版本: Windows Vagrant


3. 安装 Homestead Vagrant Box

  • 在线安装(非常耗时，请挂VPN，网速不行的直接选择离线安装)

    $ vagrant box add laravel/homestead     //根目录

  • 离线安装

    该链接附录一写的很清楚，不赘述.


4. 克隆 Homestead代码

  • gitbash克隆homestead项目

    $ git clone https://github.com/laravel/homestead.git   //从github上克隆homestead项目

  • 创建配置文件

    $ cd Homestead  //进入Homestead目录
$ init.bat  //创建配置文件


5. 编辑 Homestead.yaml 文件（根据需要添加）

  • ip(后面需要)

    ip: "192.168.10.10"

  • authorize & keys (后面需要)

    authorize: ~/.ssh/id_rsa.pub
keys: - ~/.ssh/id_rsa

    如果之前电脑没有添加过SSH 秘钥，可参考文章Github添加SSH密钥

  • provider(在安装中选3即可)

    provider: virtualbox

  • folders(共享文件)

    folders:
    - map: D:/code   //windows下绝对路径
      to: /home/vagrant/code

    需要新建code文件夹

    $ mkdir -p code   //用命令行创建

  • site(配置Nginx站点)

    sites:
    - map: homestead.test   //站点名称可修改，以.test结尾即可(.app不可用，被google买了)
    to: /home/vagrant/code/program_name/public    //对应public下的index.php,可省略


6. 编辑 hosts 文件

Windows 系统hosts文件路径：C:\Windows\System32\drivers\etc\hosts，可右键用文本打开编辑后保存。

192.168.10.10  homestead.test   //添加到hosts文件尾


7. 启动虚拟机（Homestead目录）

$ vagrant up    //直接启动
$ vagrant up --provision    //若Homestead.yml文件有改动,加上--provision

7.1 vagrant up失败。

报错如下：

There was an error while executing `VBoxManage`, a CLI used by Vagrant for controlling VirtualBox. The command and stderr is shown below.

通过以下三步解决了问题

  1. 找到vboxmanage.exe 右键属性，兼容win8运行
  2. 控制面板，程序和功能。启用或关闭windows功能，关闭Hyper-V，没有改选项，保证IIS没有开启。

  3. 开启虚拟化(win10系统默认关闭虚拟化技术

    • 任务管理器，性能，CPU可查看虚拟化是关闭状态
    • 窗口，设置，更新和安全，恢复，立即重启，疑难解答，高级选项，UEFI固件设置，重启进入BIOS，将虚拟化Enable。
    • 任务管理器，性能，CPU可查看虚拟化是开启状态
    • vagrant up 成功


8. SSH 登入虚拟机

$ vagrant ssh


9. 退出并关闭虚拟机

$ exit    //退出
$ vagrant halt    //关闭虚拟机

///////补充/////////
$ vagrant reload --provision    //重启虚拟机
$ vagrant destroy   //销毁虚拟机


10. 在浏览器输入homestead.test
