环境依赖

安装

  1. 执行git clone克隆项目
  2. 执行mv .env.XXX .env根据部署位置拷贝对应配置文件
  3. 执行sudo composer install安装PHP库依赖

  4. 执行sudo npm install安装NodeJS库依赖
