一、准备工作

  1. npm安装vue-amap

    $ npm install vue-amap --save

  2. 引入vue-amap

  3. 地图组件

    //amap.vue
<template>
    <el-amap ref="map"></el-amap>
</template>
<script>
// NPM 方式
// import { AMapManager } from 'vue-amap'
// CDN 方式
let amapManager = new VueAMap.AMapManager()
module.exports = {
    data() {
        return {
        amapManager,
        zoom: 12,
        center: [121.59996, 31.197646],
        }
    }
}
</script>

    注意：更多地图属性字段查看文档:基础-地图

二、兼容原生SDK

直接参考文档，这里没有坑：

  1. 脚本初始化
  2. 兼容高德原生SDK

三、vue-amap的踩坑

  1. 图片覆盖物

    点击事件无效，bug至今未修复，github好久没有维护了，转送门。可用原生SDK解决。

  2. 点标记

    //amap.vue
<el-amap>
  <el-amap-marker
      v-for="(marker,index) in boatMarkers"
      :key="'boat' + index"
      :visible="boatMarkersAll.visible"
      :position="marker.position"
      :events="marker.events"
      :angle="marker.angle"
      :icon="require('@/assets/icon/boat2.svg')"
  />
</el-amap>
