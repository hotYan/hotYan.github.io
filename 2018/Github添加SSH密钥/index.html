添加SSH密钥 [完结]

个人比较喜欢先甩参考链接,可以选择直接去看参考链接。

  • GitHub Help
  • 参考连接

参考链接有整体的流程总结。跟着步骤来没有问题。但我会觉得GitHub Help比较官方？权威？

一、查看现有的SSH密钥

$ ls -al ~/.ssh


二、生成新SSH密钥

  1. 根据实际情况填写GitHub电子邮件地址
    $ ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
  2. 文件生成位置
    Enter file in which to save the key (/Users/hotkang/.ssh/id_rsa): [Press enter]  
/*按回车默认文件位置*/
  3. 设置push密码
    Enter passphrase (empty for no passphrase):
Enter same passphrase again: 
/*设置push文件时的密码,回车默认没有密码*/


三、添加密钥到GitHub

  1. 打开 /.ssh/id_rsa.pub 文件,将内容复制到剪贴板
  2. 单击右上角头像
  3. 单击 Settings
  4. 单击 SSH and GPG keys
  5. 单击 New SSH Key
  6. 填写Title并将/.ssh/id_rsa.pub文件内容粘贴到Key
  7. 单击Add SSH Key完成
  8. 如果出现提示,请确认你的GitHub密码


四、测试SSH连接

$ ssh -T git@github.com
