2019.08.20

一、问题描述

npm install 报错，如下：

npm ERR! code EINTEGRITY
npm ERR! sha1-0B+mxlhZt2/PMbPLU6OCGjEdgFE= integrity checksum failed when using sha1: wanted sha1-0B+mxlhZt2/PMbPLU6OCGjEdgFE= but got sha512-HQStPIV4y3afTiCYVxirakhlCfGkI161c76kKFca7Fk1JusM//Qeo1ej2XaMniiNeaZklMVrh3vTtIzpzwbpmA== sha1-Co0Wc0odmvvIJLmd+H5zjlji2lk=. (9418 bytes)

二、参考链接

  1. integrity checksum failed when using sha1 - Node v8+ #13635
  2. npm ERR! code EINTEGRITY 解决方案

三、解决办法

  1. 运行参考链接1中的高赞

    $ npm install -g npm
#结果如下：
/Users/hotkang/.nvm/versions/node/v10.15.3/bin/npm -> /Users/hotkang/.nvm/versions/node/v10.15.3/lib/node_modules/npm/bin/npm-cli.js
/Users/hotkang/.nvm/versions/node/v10.15.3/bin/npx -> /Users/hotkang/.nvm/versions/node/v10.15.3/lib/node_modules/npm/bin/npx-cli.js
+ npm@6.10.3
updated 1 package in 11.68s

    注：npm install 依旧不成功

  2. 删文件

    删除package-lock.json、node_modules文件,清缓存：

    $ rm ./package-lock.json
$ rm -r ./node_modules
$ npm cache clear --force 
#报没有权限，根据提示给权限
# To permanently fix this problem, please run:
# npm ERR!   sudo chown -R 501:20 "/Users/hotkang/.npm"
#给权限
$ sudo chown -R 501:20 "/Users/hotkang/.npm"
#再次清缓存
$ npm cache clear --force

    注：npm install 成功！
