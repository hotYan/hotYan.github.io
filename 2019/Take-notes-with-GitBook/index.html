一、准备工作

  1. 默认已安装Node.js

    没有安装node运行环境？请参考用nvm装node.js。

  1. 默认会用Markdown

    不会编辑 .md 文件？请参考Markdown。


二、本地运行

  1. 全局安装

    $ npm install gitbook-cli -g   //在你想放置项目的目录下运行

  2. 查看版本

    $ gitbook -V 
//我的版本
CLI version: 2.3.2
GitBook version: 3.2.3

  3. 初始化gitbook

    $ gitbook init

info: create README.md 
info: create SUMMARY.md 
info: initialization is finished

  4. 启动 gitbook 本地服务器

    $ gitbook server  //浏览器访问http://localhost:4000




三、将 GitBook 部署到 GitHub

将编译后的 _book 目录 push 到远程仓库

很简单，具体步骤参考之前写的在GitHub Pages上部署自己的简历。
